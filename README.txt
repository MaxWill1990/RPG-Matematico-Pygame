RPG Matemático | Projeto de Python & Pygame
Este jogo nasceu de um desafio real durante as minhas aulas de ADS (Análise e Desenvolvimento de Sistemas). O professor apresentou a biblioteca Pygame e nos ensinou a criar um jogo de nave, mas como eu sempre fui apaixonado pelos RPGs de turno dos anos 90, decidi ir um passo além e criar algo com a minha cara.

🎮 O Jogo
O conceito é simples, mas desafiador: o seu conhecimento é o seu poder de ataque.

Acertou o cálculo matemático? Você dá dano no inimigo.

Errou a conta? O monstro contra-ataca e você perde HP.

Foi a forma que encontrei de unir lógica de programação, matemática e diversão em um projeto só.

🛠️ Desafios e "Bugs" da Vida Real
Quem olha o código pronto não imagina a correria que foi. Tive pouco tempo para entregar e enfrentei problemas que me fizeram quebrar a cabeça, como:

O sumiço das fontes: Em vários momentos, o texto da interface simplesmente desaparecia da tela por causa da ordem de renderização.

Inimigos fantasmas: Bugs na lógica de "spawning" que faziam o inimigo não aparecer quando devia.

Ajuste de áudio: Sincronizar os sons de acerto e erro sem travar o loop principal do jogo.

🤖 Uso de IA no Projeto
Neste projeto, usei o Gemini como meu "parceiro de equipe". Ele me ajudou na criação dos assets (imagens e sons) e foi fundamental para me explicar a estrutura de alguns blocos de código que eu ainda não dominava.

O maior aprendizado aqui foi entender que a IA é uma ferramenta poderosa, mas ela não faz o trabalho sozinha. Eu precisei entender o "porquê" de cada linha de código para conseguir debugar e alinhar todo o ecossistema do game.

🚀 Como rodar o projeto
Tenha o Python instalado na sua máquina.

Instale a biblioteca Pygame: pip install pygame.

Clone o repositório e rode o arquivo principal: python main.py.
